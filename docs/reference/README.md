# AWS Reference Files

This folder contains original AWS/EKS reference files from the upstream project.

They are kept for future cloud deployment reference only.

The completed implementation in this fork is No-AWS and runs locally using:

- Docker Compose
- GitHub Actions Local No-AWS CI
- Minikube
- Argo CD
- Prometheus/Grafana
- Local AIOps health analyzer

The active GitHub Actions workflow is:

.github/workflows/local-ci.yml
